A subquery is a query within another query

first inner query run, give result to out query, then outer query run

inner query scope is till outer query run


Types :-

Based on What data it return

1. Scalar - return single value , eg 9.3

2. Row Query - will return multiple rows but one column , Gener - Action,Comedy, Adventure

3. Table Query - return table with multiple column and rows

Based on working

1. Independent - inner query not depend on outer query

2. Co-related - inner query depend on outer query (bahar vale query ka data chahiye to run)
